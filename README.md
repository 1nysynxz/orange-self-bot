# orange-self-bot
Orange-self-bot
🍊 ORANGE SELF-BOT
​A high-performance, multi-functional Discord self-bot designed for speed, utility, and account automation. Built to run 24/7 on hosting platforms like Heavencloud.
​🚀 Features
​🛡️ Utility & Intelligence
​IP Lookup: High-speed IP tracing with ISP details and Google Maps integration.
​Dynamic Help: ANSI-colored, multi-page help menu with anti-crash protection.
​Auto-Quest: 24/7 Discord Orb farming. Spoofs game presence for Genshin, Honkai, Valorant, and more.
​Game Spoofer: Set custom "Playing" statuses with optional 24/7 heartbeat to prevent status drops.
​🤖 Automation
​Auto-Messager (AM): Instantly respond to specific users with randomized roasts from roasts.txt.
​Fast Roasting: Mass-roast targets with memory-cached text files for zero-latency replies.
​💀 Destruction (Use Wisely)
​Nuke: Instant channel deletion and mass-creation.
​Ban All: Asynchronous multi-threaded banning (Super Fast).
​Role Wipe: Mass deletion of server roles.
​🛠️ Installation
clone this repo git clone [https://github.com/1nysynxz/orange-self-bot.git](https://github.com/1nysynxz/orange-self-bot.git)
